   This document presents two different types of computer architecture: Von Neumann's and Harvard architectures. And shows differences between them. For viewing the presentation please install Vostan Office.

   
   Installation process
   
   1. Download Vostan Office from https://ggg.vostan.net/#67 (appropriate version 
      for your OS)
   2. Install from command line (for Linux (Debian Core): 
      dpkg -i [vostan_office_*.deb]
